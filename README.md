## Run Locally

**1. Clone**
```bash
git clone https://github.com/Muktadirul675/as-safa-interview-task1.git && cd as-safa-interview-task1
```

**2. Install packages**
```bash
npm install
```

**3. Run dev server**
```bash
npm run dev
```

**4. Run production server**
```bash
npm run build
npm run preview
```
